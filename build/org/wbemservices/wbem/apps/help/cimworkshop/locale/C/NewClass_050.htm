Class Qualifiers

Click this button to create qualifiers for a new subclass. When you click this button, the Qualifiers dialog box is displayed, which you use to specify values, including scope and flavor for the qualifier.

To Add Qualifiers

  1. In the Qualifiers dialog box, right click the Qualifier for which you want to set new values and select Add Qualifier.

    The Add Qualifier dialog box is displayed.

  2. In the Add Qualifier dialog box, select the name of a qualifier in the list that you want and click OK.

    The New Class dialog box is displayed.
