Qualifiers

Use this dialog box to view the qualifiers of the property, method, or class that you select.

To display the Scope dialog box, click the Scope button on the Qualifiers dialog box. In the Scope dialog box, you can view the scope of a qualifier.

To display the Flavors dialog box, click the Flavors button in the Qualifiers dialog box. In the Flavors dialog box, you can view the potential uses, or flavor, of a qualifier.
