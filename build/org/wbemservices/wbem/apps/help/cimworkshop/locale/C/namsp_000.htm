Change Namespace

In the Java WBEM SDK, the default namespace is root\cimv2. Use this dialog box to change to another namespace in which to work. Change namespaces when you want to use unique classes that are located in another namespace. Also use this dialog box to create or delete a namespace.

To Move to Another Namespace: Menu Selection or Icon Method

  1. In the Namespace window, select the namespace that you want to use.

  2. Click OK.

    The namespace that you selected becomes the current namespace.

To Create a Namespace

  1. Click Add.

    The New Namespace dialog box appears.

  2. Type the name of the new namespace that you want to create and click OK.

    The new namespace that you specified is created in the namespace that you selected.

To Delete a Namespace

  1. In the Namespace window, select the namespace that you want to delete.

  2. Click Delete.

    A prompt appears that asks you to confirm that you want to delete the namespace that you selected.

  3. Click OK.

    The namespace that you selected is deleted.
