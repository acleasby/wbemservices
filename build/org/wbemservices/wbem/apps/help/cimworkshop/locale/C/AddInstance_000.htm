Add Instance

To create a new instance of the selected class, you must define values for each of the key properties shown in the table.

If you are allowed to change a property, click in the Value field to open a Set value dialog box.

When you add an instance, you set the key property values of that instance. You cannot change the values of inherited properties.

To Add an Instance to a Class

  1. Click in the Value field for the instance key property that you want.

    The Set value dialog box is displayed.

  2. In the Value field, type the value that you want and click OK.

    The Add Instance dialog box is displayed.

  3. Repeat the previous two steps until you've set all the values in the Value column.

  4. On the Add Instance of class dialog box, click OK.

    The Instance Editor Window is displayed.
