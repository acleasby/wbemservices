Reference

Use this dialog box to edit the reference property of an association class.

When you set the value of a reference to a class, you are setting an instance. On the Set value dialog box, you can choose an instance of that class or you can manually set the values of the keys.

A reference property is a pointer to another class (or instance).

For each instance of the referred-to class, you can either:

  • Directly edit the value of the key properties
  • Select an instance of the referred-to class, and in doing so, define the key properties from that instance.

To directly edit the value of any key property, click the key property in the column under Value.

Note: If you are creating a static association instance that you want to store in the repository, the reference itself must point to a static instance in the repository.
