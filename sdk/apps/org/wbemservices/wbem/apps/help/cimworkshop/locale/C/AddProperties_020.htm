Type

In the Type selection list, select the type of property that you want.

Property types that you can specify include unsigned integer, signed integer, string, Boolean, real number, datetime, character, array, and reference.

A reference property only exists in an association class.

When you select REF as the property type and click OK, the Class Selection dialog box is displayed. You use the Class Selection dialog box to select the class to which you want the reference property (REF) to point.
