Host URL

Use this field to specify the URL of the host on which the CIM Server is running and that contains the namespace in which you want to work

You can also use this field to specify the protocol you connect with, the namespace you want to connect to and the port you want to connect on

Example: If you wanted to connect to the namespace root/cimv2 on the host named crawfish using http on port 5988, you would specify the host as:

http://crawfish:5988/root/cimv2
