How to Run MOF2HTML on Win32

Last Updated 12/19/03

mof2html is a command that generates HTML files for the CIM classes defined in a MOF file. WARNING: The MOF file should include any Qualifier defintions that its class definitions require. See Simple.mof

  • Before running mof2html.bat, you must first install Xalan-Java v2.3.1 XSLT processor. Download xalan-j_2_3_1-bin.zip and unzip into c:\xalan-j_2_3_1. NOTE: If you install into another directory or have a different version installed, you must edit the XALAN_HOME variable in mof2html.bat
  • Open a command window (DOS window).
  • Go to the directory containing the WBEM Services executables.
    • If you built the WBEM Services executables in c:\files\open, go to directory c:\files\open\dist\wbemservices\bin.
    • If you downloaded or copied the WBEM Services executables into c:\wbemservices, go to directory c:\wbemservices\bin
    c:\>cd c:\wbemservices\bin
  • Run mof2html.bat. You will see the following:
